### Структура папки `/gopkg`

| Папка | Назначение |
|-------|-------------|
| **/auth**        | Библиотеки для аутентификации через TVM |
| **/backoff**     | Утилиты для реализации механизмов повторных попыток с экспоненциальной задержкой |
| **/contest**     | HTTP‑клиент для работы с сервисом **Yandex Contest** |
| **/datum**       | HTTP‑клиент для работы с сервисом **Datum** |
| **/environment** | Определение типов окружения (`prod`, `test`, `dev`) |
| **/envutils**    | Утилиты для работы с переменными окружения |
| **/grpcutils**   | Вспомогательные функции для работы с **gRPC** (обработка ошибок, заголовков) |
| **/logger**      | Настройка и управление логированием |
| **/errorbooster**| Компонент для отправки ошибок в систему мониторинга |
| **/utils**       | Вспомогательные функции для работы с атрибутами логов |
| **/metrics**     | Инструменты для сбора и экспорта метрик сервиса |
| **/organizer**   | HTTP‑клиент для работы с сервисом **Organizer** (организация учебного процесса) |
| **/parseutils**  | Утилиты для парсинга и преобразования различных типов данных |
| **/s3**          | Библиотека для работы с **S3** |
| **/postgres**    | Библиотека для работы с **PostgreSQL** |
| **/saas**        | HTTP‑клиент для работы с **Yandex SaaS** |
| **/store**       | HTTP‑клиент для работы с сервисом **Store** (стор задач в образовании) |
| **/telemost**    | HTTP‑клиент для работы с **Yandex Telemost** |
| **/traces**      | Инструменты для трассировки запросов |
| **/tracker**     | Компоненты для отслеживания и мониторинга HTTP‑запросов |
